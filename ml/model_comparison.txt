Review Classification — Model Comparison
=========================================

CONTEXT
-------
Two models were built iteratively. V1 established a baseline. V2 addressed
identified weaknesses. V2 is the final model used in the PraxisIQ dashboard.

All numbers below are on the same held-out test set:
  300 reviews → 240 train / 60 test (stratified 80/20 split, random_state=42)

─────────────────────────────────────────────────────────────────────
                        MODEL V1          MODEL V2 (FINAL)
─────────────────────────────────────────────────────────────────────
Vectorization     TF-IDF (unigrams)   TF-IDF (unigrams + bigrams)
max_features             5000                    5000
Class balancing          None              class_weight='balanced'
Regularization     Default (C=1.0)          Default (C=1.0)
max_iter                  100                    2000
─────────────────────────────────────────────────────────────────────
Accuracy                68.33%                  83.33%
Macro avg F1             0.57                    0.78
Weighted avg F1          0.65                    0.82
─────────────────────────────────────────────────────────────────────

WHY V2 OUTPERFORMS V1
----------------------
1. Bigrams captured multi-word clinical phrases that unigrams missed:
   - "waiting time", "root canal", "follow up", "cleaning service"
   These phrases are category-specific and invisible to unigram tokenization.

2. Class balancing corrected the Positive-class dominance.
   V1 classified ambiguous reviews as Positive by default.
   V2 applies inverse-frequency weights so minority classes have equal pull.

3. max_iter=2000 ensured convergence — V1's default 100 iterations
   showed convergence warnings for some random seeds.

REMAINING WEAKNESSES IN V2
---------------------------
- Communication (F1: 0.57): Overlaps heavily with Staff and Neutral.
  Reviews like "the receptionist didn't call back" could be either.
- Neutral (F1: 0.50): Only 18 samples in the full 300-review dataset.
  Too few training examples for reliable Neutral detection.

These categories are where LLM classification adds the most value —
see llm/llm_evaluation_final.py for prompt-based comparison.

CONCLUSION
----------
Model V2 selected as final model. 83.33% accuracy on 60-review test set.
Script: ml/review_classifier_v2.py

Review Classification — Model Results Summary
=============================================

TWO MODELS WERE BUILT AND COMPARED. See model_comparison.txt for full comparison.

─────────────────────────────────────────────
MODEL V1 — Baseline (Rejected)
─────────────────────────────────────────────
Vectorization : TF-IDF (unigrams only, max_features=5000)
Model         : Logistic Regression (default, no class balancing)
Dataset       : 300 reviews → 240 train / 60 test (80/20 split, stratified)
Accuracy      : 68.33%

Issues identified:
- No class balancing → Positive class dominated predictions
- Unigrams only → missed multi-word patterns (e.g. "waiting time", "root canal")
- Neutral category F1 near zero due to only 18 samples in full dataset

─────────────────────────────────────────────
MODEL V2 — Final Selected Model
─────────────────────────────────────────────
Vectorization : TF-IDF (unigrams + bigrams, max_features=5000)
Model         : Logistic Regression (max_iter=2000, class_weight='balanced')
Dataset       : 300 reviews → 240 train / 60 test (80/20 split, stratified)
Accuracy      : 82.22%

Per-class results:
  Positive      → Precision: 0.84  Recall: 0.95  F1: 0.89  (support: 22)
  Pricing       → Precision: 0.88  Recall: 1.00  F1: 0.93  (support: 7)
  Staff         → Precision: 1.00  Recall: 0.83  F1: 0.91  (support: 6)
  Waiting Time  → Precision: 0.78  Recall: 1.00  F1: 0.88  (support: 7)
  Treatment     → Precision: 0.83  Recall: 0.71  F1: 0.77  (support: 7)
  Communication → Precision: 0.67  Recall: 0.50  F1: 0.57  (support: 8)
  Neutral       → Precision: 1.00  Recall: 0.33  F1: 0.50  (support: 3)

Weighted average:
  Precision: 0.84  Recall: 0.83  F1: 0.82

Improvements over V1:
- +15% accuracy (68.33% → 82.22%)
- Class balancing corrected minority class suppression
- Bigrams captured multi-word treatment and complaint phrases
- Better recall on Communication, Treatment, Staff categories

Hardest categories: Communication (F1: 0.57), Neutral (F1: 0.50)
Reason: Semantic overlap — same words appear across Communication, Staff,
and Neutral reviews. These categories benefit most from LLM classification.
